Developer Tools

Developer tools, commonly known as Deve Tools, are a set of built-in tools avaliable in web browsers like (Chrome, Firefox, and Edge) that allow web developers to inspect, debug and analyze web applications. These features that Dev Tools offer are essential for web developers to build and maintain websites.

Elements Panel

The Elements panel allows you to inspect, edit, and modify HTML & CSS in real time. It lets you experiment with styles wihtout changing the actual website code. It also allows you to view the DOM (Document Object Model) of the webpage, which is the structure of the HTML elements on the page.

Network Panel

The Network panel shows all network requests mady by the page(HTML, CSS, JavaScript, Images,applications) It measure how long each resource take to load.It also helps a Web Developer analyze page load times and performance issues and focus on specific resource tyesp (XHR, CSS, JS, Media,etx.),

Slow 3G Throttling

Throttling to Slow 3G this is done by the Web Developer intentional reduction of data speeds by webpage slow 3g throttling is more about how users experience the websit's performance under defferent network conditions such as when a user's connections is throttled to slower speeds like 3G. It affects how quickly web pages load and how responsive the site is, and how smoothly multimedia elements like videos and or images display. Throttling to Slow 3G helps you understnd how you webpage performs for users on slower networks

Questions & Answers (Q&A)

Question 1: Which panels in Developer Tools (Deve Tools) will you use to inspect HTML elements and change? modify their style properties?

Answer: The Elements Panel where you can inspect and modify the HTML structure and CSS styles oof your webpage. In the elements panel, you can see and modify the CSS rules applied to the selected element in real-time. Live editing you can use the Console panel to write and execute JavaScrip code, which can dynamically change the style or content of you webpage.

Question 2: Which panel do you use to inspect network activity?

Answer: The Network Panel provides detailed information about all network requests made by you webpage including resource loading times, request headers response header, and more. The Network panle displays a list of all network requests, including files like HTML, CSS, JavaScript, Images, and toher resources

Question 3: Why would you Throttle to Slow 3G?

Answer: To mimic slow internet speeds to test website performance. Therefore observing page loading behavior under limited bandwith. Identify slow-loading elements (large images, unoptimized scripts). and to improve user experience for mobile user with poor connections.

In Summary

The Elements Panel in DevTools allows you to inspect and modify HTML and CSS in real time, making it usful for testing style changes and layout adjustments without altering the actual code. The Newtwork Panel helps monitor all requests made by a webpage, including HTML, CSS, JavaScript, images,and API calls. It enables developers to check load times, filter specific request types, and inspect headers and responses for debugging. Additionally, Slow 3g Throttling simulates a slow internet connection, allowing developers to analyze page performance under limited bandwidth, identify boottlenecks, and optimize loading strategies for a better user experience. Together, these tools help developers inspect, debug, and optimize webpages effectively.
